# Pace Analysis Project

## Overview 
This project analyzes pace changes in a 5 kilometer run over time using Excel 

## Features  
- Pace tracking across multiple dates spanning 7 months
- Use of Pivot table for multiple statistical measures
- Data Visualization using graphical tools
- Monthly analysis
- Trend analysis
- Deliverables as a tracking measure of progress

## Tools Used 
- Microsoft Excel
- Pivot table
- Lookup formulas
- Conditional Functions (IF/IFS, SUMIFS, MINIFS/MAXIFS, COUNTIFS)
- Text Editing Functions (Concatenate, right/left/mid, etc.)
- Statistical Functions (COUNT, SUM, MIN/MAX, AVERAGE, STDEV.S, Median, etc.)
- Cell formatting

## Key Insights 
- Identified running average pace trend over time
- Compared monthly performances using comprehensive metrics such as pace consistency and relative pace consistency
- Identified patterns leading to faster paces
- Discovered key measures affecting running performance
